Save the content of a vector to a file. To later retrieve the content you should use the function vector_load_content.

Parameters
arrayThe vector to save
fileThe file to save to
save_funcThe function that will be applied to each element of the vector. This function will write the content of an element of the vector at its own discretion.
Note
save_func will write the elements, remember to create an equivalent of this function for vector_load_content to load an element using the same data scheme.
Returns
1 on error, 0 otherwise
