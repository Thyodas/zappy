◆ map_distribute_resources()

void map_distribute_resources ( map_t *  map,
resource_t *  density 
)

Distribute resources across the map.

This function distributes different types of resources across the map based on the map's dimensions and specified resource densities. It calculates the quantity of each resource type based on the map's size and the corresponding density values. Then, it iterates over each row of the map and distributes the resources using the row_distribute_resources function until all resources have been distributed.

Parameters
mapThe map structure to distribute resources on.
densityThe number of each resource to be distributed.
