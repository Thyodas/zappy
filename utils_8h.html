char ** split_str_to_array ( char *  str,

int *  len,
int  max_arg 
)

Splits a string into an array of substrings based on the specified separators.

Parameters
strThe string to be split.
separatorsA string containing the characters to be used as separators.
lenPointer to an integer variable to store the length of the resulting array.
max_argMaximum number of substrings to be extracted. If the number of substrings reaches or exceeds this value, the remaining separators will be ignored.
Returns
A dynamically allocated array of strings, where each element is a substring from the original string. The last element of the array is set to NULL.
Note
It is the responsibility of the caller to free the memory allocated for the array.
Warning
Strings are not malloced, they are pointers from the str, like strtok.
